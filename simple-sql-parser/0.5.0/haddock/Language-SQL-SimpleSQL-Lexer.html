data Token

Represents a lexed token

Constructors

Symbol String

A symbol is one of the following

  • multi char symbols <> == != ||
  • single char symbols: * + - ^ / % ~ & | ? ( ) [ ] , ; ( )
Identifier String

This is an identifier or keyword.

QIdentifier String

This is an identifier quoted with "

UQIdentifier String

This is an identifier quoted with u&"

DQIdentifier String String String

This is a dialect specific quoted identifier with the quote characters explicit. The first and second fields are the starting and ending quote characters.

HostParam String

This is a host param symbol, e.g. :param

SqlString String

This is a string literal.

CSSqlString String String

This is a character set string literal. The first field is the character set (one of nNbBxX, or u&, U&).

SqlNumber String

A number literal (integral or otherwise), stored in original format unchanged

Whitespace String

Whitespace, one or more of space, tab or newline.

LineComment String

A commented line using --, contains every character starting with the '--' and including the terminating newline character if there is one - this will be missing if the last line in the source is a line comment with no trailing newline

BlockComment String

A block comment, /* stuff */, includes the comment delimiters
